# 🐶 PetShop - PHP

Simple PHP project using MySQL to create, list, edit, and delete **pets** and **species**.

## Features

- Create, list, edit, and delete species
- Create, list, edit, and delete pets
- User registration
- Simple session-based login

## Technologies

- PHP
- MySQL
- HTML/CSS/JavaScript
